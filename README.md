# ☕ Cafe Sales Analysis Dashboard

An interactive **Microsoft Excel Dashboard** built to analyze coffee shop sales data and uncover business insights using **Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Excel formulas**.

---

## 📊 Dashboard Preview

![Dashboard](Images/Dashboard.png)

---

## 🎯 Project Objective

Transform raw sales data into an interactive dashboard that enables users to:

- Monitor overall sales performance
- Analyze customer footfall
- Identify peak ordering hours
- Compare store performance
- Track top-selling products
- Explore sales using interactive slicers

## 📌 Dashboard Features

- Interactive Month & Day slicers
- Quantity Ordered by Hour
- Sales Distribution by Category
- Order Size Distribution
- Sales & Footfall by Store Location
- Top 5 Products by Sales
- Orders by Weekday

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- GETPIVOTDATA
- Data Visualization

---

## 💡 Key Insights

- Morning hours (8–10 AM) record the highest order volume.
- Coffee contributes the largest share of total sales.
- Hell's Kitchen generates the highest sales among all stores.
- Barista Espresso is the highest-selling product.
- Friday experiences the highest customer orders.
